< 과제2 클래스와 인스턴스 개념 > 2줄이상
class
객체를 만드는 틀, 내부에 메소드를 포함
매직메소드중 __call__이라는 메소드가 있는데
함수를 정의하고 ( ) 로 함수를 호출했었는데
사실 ( ) 는 __call__이라는 매직 메소드를 의미하는 것이었음

instance
객체라고도 하며, 틀에 의해 생성된 것
객체에 . 을 찍으면 해당 객체에 접근할 수 있었는데,
이것도 사실 매직메소드 덕분임
__getattribute__ 라는 이름의 매직메소드를 호출하는 것.

>>> class Foo:
        def func1():
                print("function 1")
        def func2(self):
                print("function 2")
>>> f = Foo()
>>> f.func2()
function 2
>>>

Foo는 class
def func1():
                print("function 1")
        def func2(self):
                print("function 2")
는 메소드
f는 인스턴스
. 으로 (getattribute매직메소드) 클래스를 호출해서 ( ) 로 (call매직메소드)로 메소드에 접근
