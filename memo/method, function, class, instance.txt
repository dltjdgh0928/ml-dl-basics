< 과제 1 메서드와 함수의 차이 > 3줄이상
method(클래스 함수)
클래스 내부에 선언되어 있는 함수
함수라고 봐도 무방하나 단지 위치만 클래스 안에 정의 되어 있으면 method
그중, __로 시작해서 __로 끝나는 메소드를 특별히 매직 매소드라고 지칭함

fucntion 
독립적으로 존재하는 것
메서드는 함수의 부분집합이라고 볼 수 있음(함수가 더 넓은 개념)

< 과제2 클래스와 인스턴스 개념 > 2줄이상
class
틀, 내부에 메소드를 포함
매직메소드중 __call__이라는 메소드가 있는데
함수를 정의하고 ( ) 로 함수를 호출했었는데
사실 ( ) 는 __call__이라는 매직 메소드를 의미하는 것이였음

instance
객체라고도 하며, 틀에 의해 생성된 것
객체에 . 을 찍으면 해당 객체에 접근할 수 있었는데,
이것도 사실 매직메소드 덕분임
__getattribute__ 라는 이름의 매직메소드를 호출하는것.
